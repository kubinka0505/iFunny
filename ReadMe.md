## Installation & Usage üìù

1. Clone the repository.
	```bash
	git clone https://github.com/kubinka0505/iFunny-Captions
	cd iFunny-Captions
	```
2. Install requirements from `requirements.txt` by `python -m pip install -r requirements.txt`
3. Move [the required files](https://github.com/kubinka0505/iFunny-Captions/ReadMe.md#requirements-) to repository folder (`__init__.pyw` location) or to any environment variables.
4. Modify the parameters in the `Config.json`. [Its documentation can be found here](https://github.com/kubinka0505/iFunny-Captions/wiki/Configuration-Documentation).
5. Open `Run.sh`
...or go to Your command prompt and type the following: `python __init__.pyw`
6. Share Your image from the `Images` directory.
