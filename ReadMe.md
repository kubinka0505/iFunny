## Requirements 📥
Modules:
- `Python >= 3.6`
- `Pillow >= 5.1.0`
- `requests >= 2.12.5`
- `emoji >= 0.4.5`
- `clipboard >= 0.0.4`
- `auepa` *(optional)*

Packages:
- **`FFmpeg >= 4.2.0 static executable binary / package`** - Since `PIL.ImageSequence.Iterator` messes up the frames colors.
  - [Windows](https://videohelp.com/software/ffmpeg/old-versions)
  - [Linux](https://johnvansickle.com/ffmpeg/old-releases)
- **`GifSicle >= 1.92-2 static executable binary / package`**
  - [Windows](https://eternallybored.org/misc/gifsicle/releases)
  - [Linux (Debian)](https://packages.debian.org/unstable/graphics/gifsicle)
- **`PNGQuant static executable binary / package`** *(optional)*
  - [Windows](https://web.archive.org/web/*/https://pngquant.org/pngquant-windows.zip)
  - [Linux (Debian)](https://packages.debian.org/search?keywords=pngquant)

## Installation & Usage 📝

1. Clone the repository and move to its directory.
	```bash
	git clone https://github.com/kubinka0505/iFunny-Captions
	cd iFunny-Captions
	```
2. Install requirements from `requirements.txt` by inputting `python -m pip install -r requirements.txt`
3. Move [the required files](https://github.com/kubinka0505/iFunny-Captions#requirements-) to repository folder (`__init__.pyw` location) or allocate them to `PATH` system environment variable.
4. Modify the parameters in the `Config.json`. [Its documentation can be found here](https://github.com/kubinka0505/iFunny-Captions/wiki/Configuration-Documentation).
5. Open shell script file named `Run` (extension is system dependent) or input `python __init__.pyw` command.
6. Share Your image from the `Images` directory.
